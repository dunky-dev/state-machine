# API

The public exports of `@chimba-ui/state-machine`.

- [`machine()`](/api/machine) — define a machine.
- [`setup()`](/api/setup) — typed machine factory.
- [`connector()`](/api/connector) — turn state into bindings.
- [`compose()`](/api/compose) — combine machines.
- [`createStore()`](/api/create-store) — the store primitive.
- [`makeReaction()`](/api/make-reaction) — react to state changes.
- [Guards](/api/guards) — `and`, `or`, `not`.
- [Actions](/api/actions) — `act`, `oneOf`.
